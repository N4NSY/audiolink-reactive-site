AudioLink Reactive

AudioLink Reactive は、VRChatアバター向けの Unity Editor ツールです。lilToon の AudioLink 表現を専用Windowで扱いやすくし、Avatarを指定して自動セットアップしたあと、Renderer一覧から対象を選んで Material 単位で調整できます。「光らせる」「膨らませる」に対応し、Preview で見た目を確認しながら設定できます。

AudioLink Reactive is a Unity Editor tool for VRChat avatars. It simplifies lilToon AudioLink workflows in a dedicated window: assign an avatar, run auto setup, select targets from the renderer list, and tune each material independently. It supports both “Glow” and “Inflate” reactions, with Preview to confirm visuals while editing.
