AudioLink Reactive

AudioLink Reactive は、lilToon の AudioLink 設定を扱いやすくする Unity Editor ツールです。複数Rendererをまとめて管理し、マテリアル単位で非破壊に調整できます。

AudioLink Reactive is a Unity Editor tool that makes lilToon AudioLink setup easier. Manage multiple renderers and tune settings per material in a non-destructive workflow.
