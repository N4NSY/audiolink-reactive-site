導入手順 Installation Steps

BOOTHからUnityPackageをダウンロードし、対象のUnityプロジェクトへimportします。

Download the UnityPackage from BOOTH, then import it into your target Unity project.

BOOTHから導入する

Install from BOOTH

  • Step 1: 「BOOTHでダウンロード」ボタンを押してBOOTHページを開きます。
  • Step 1: Click “Download from BOOTH” to open the BOOTH page.
  • Step 2: BOOTHからAudioLink ReactiveのUnityPackageをダウンロードします。
  • Step 2: Download the AudioLink Reactive UnityPackage from BOOTH.
  • Step 3: 対象のAvatar ProjectをUnityで開きます。
  • Step 3: Open your target Avatar Project in Unity.
  • Step 4: ダウンロードしたUnityPackageをUnityプロジェクトにimportします。
  • Step 4: Import the downloaded UnityPackage into your Unity project.
  • Step 5: Unityで Tools / N4NSY / AudioLink Reactive から起動します。
  • Step 5: In Unity, open Tools / N4NSY / AudioLink Reactive.

必要環境

Requirements

  • Unity: 2022.3.22f1
  • VRChat SDK - Avatars: 3.10.x
  • lilToon: 2.3.x
  • NDMF: 1.11.x
  • Modular Avatar: 1.16.x
補足 Note

UnityPackageをimportする前に、VRChat SDK、lilToon、NDMF、Modular Avatarを必要に応じて先に導入してください。

Before importing the UnityPackage, install VRChat SDK, lilToon, NDMF, and Modular Avatar as needed.

注意事項 Notes

現在は正式リリース前の開発版です。導入前に必ずUnityプロジェクトのバックアップを作成してください。

This is a pre-release development version. Please back up your Unity project before installation.

開発版について

Development Version

  • 正式リリース前: 今後の更新で仕様や設定形式が変わる可能性があります。
  • Pre-release: Settings and behavior may change in future updates.
  • バックアップ推奨: 導入前にUnityプロジェクトを複製しておくことを推奨します。
  • Backup recommended: Duplicate your Unity project before installation.
  • 動作確認: 導入後はPreviewとBuild / Uploadの確認を行ってください。
  • Check behavior: After installation, test Preview and Build / Upload.

困ったとき

Troubleshooting

  • BOOTHページが開けない: ブラウザでBOOTHページを直接開き直してください。
  • BOOTH page does not open: Open the BOOTH page directly in your browser.
  • Unityでエラーが出る: 依存パッケージの導入状況を確認してください。
  • Unity errors: Check that all dependency packages are installed.
  • 使い方が分からない: Docsページを確認してください。
  • Need help: See the Docs page.
