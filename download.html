導入手順 Installation Steps

VCCにRepositoryを追加した後、対象プロジェクトへAudioLink Reactiveを追加します。

Add the repository to VCC, then install AudioLink Reactive into your target project.

VCCから追加する

Install from VCC

  • Step 1: 「VCCに追加」ボタンを押します。
  • Step 1: Click the “Add to VCC” button.
  • Step 2: VCCが開いたら、Repositoryの追加を許可します。
  • Step 2: When VCC opens, allow it to add the repository.
  • Step 3: 対象のAvatar Projectを開きます。
  • Step 3: Open your target Avatar Project.
  • Step 4: Manage ProjectからAudioLink Reactiveを追加します。
  • Step 4: Add AudioLink Reactive from Manage Project.
  • Step 5: Unityで Tools / N4NSY / AudioLink Reactive から起動します。
  • Step 5: In Unity, open Tools / N4NSY / AudioLink Reactive.

必要環境

Requirements

  • Unity: 2022.3.22f1
  • VRChat SDK - Avatars: 3.10.x
  • lilToon: 2.3.x
  • NDMF: 1.11.x
  • Modular Avatar: 1.16.x
補足 Note

VCC側で依存パッケージが自動解決されない場合は、VRChat SDK、lilToon、NDMF、Modular Avatarを先に導入してください。

If VCC does not resolve dependencies automatically, install VRChat SDK, lilToon, NDMF, and Modular Avatar first.

開発版について

Development Version

  • 正式リリース前: 今後の更新で仕様や設定形式が変わる可能性があります。
  • Pre-release: Settings and behavior may change in future updates.
  • バックアップ推奨: 導入前にUnityプロジェクトを複製しておくことを推奨します。
  • Backup recommended: Duplicate your Unity project before installation.
  • 動作確認: 導入後はPreviewとBuild / Uploadの確認を行ってください。
  • Check behavior: After installation, test Preview and Build / Upload.

困ったとき

Troubleshooting

  • VCCに表示されない: Repository追加後、VCCを再起動してください。
  • Not visible in VCC: Restart VCC after adding the repository.
  • Unityでエラーが出る: 依存パッケージの導入状況を確認してください。
  • Unity errors: Check that all dependency packages are installed.
  • 使い方が分からない: Docsページを確認してください。
  • Need help: See the Docs page.
