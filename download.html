導入後の流れAfter Installation

導入後は、UnityのToolsメニューからAudioLink Reactiveを開いて設定します。

After installation, open AudioLink Reactive from the Unity Tools menu and start configuring your Avatar.

  1. 1. Unityで対象Avatarのプロジェクトを開く
  2. 1. Open the Unity project that contains your target Avatar
  3. 2. Tools / N4NSY / AudioLink Reactive を開く
  4. 2. Open Tools / N4NSY / AudioLink Reactive
  5. 3. Avatar欄に対象アバターを指定する
  6. 3. Assign your target avatar to the Avatar field
  7. 4. レンダラー一覧から対象Rendererを追加する
  8. 4. Add target renderers from the Renderer list
  9. 5. Materialごとに「光らせる」「膨らませる」を設定する
  10. 5. Configure Glow or Inflate for each material
