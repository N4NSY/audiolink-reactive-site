パッケージ名 Package Name

com.n4nsy.audiolink-reactive

導入手順 Installation Steps

  1. 上の「VCCに追加」ボタンを押します。
  2. VCCが開いたら、Repositoryの追加を許可します。
  3. 対象のAvatar Projectを開きます。
  4. Manage ProjectからAudioLink Reactiveを追加します。
  5. Unityを開き、Tools / N4NSY / AudioLink Reactive から起動します。
  1. Click the “Add to VCC” button above.
  2. Allow VCC to add the repository.
  3. Open your Avatar Project.
  4. Add AudioLink Reactive from Manage Project.
  5. Open Unity and launch it from Tools / N4NSY / AudioLink Reactive.

必要環境 Requirements

  • Unity 2022.3.22f1
  • VRChat SDK - Avatars 3.10.x
  • lilToon 2.3.x
  • NDMF 1.11.x
  • Modular Avatar 1.16.x
