はじめかたGetting Started

AudioLink Reactive は、lilToon の AudioLink 設定をまとめて扱いやすくするための Unity ツールです。まずは Controller を追加し、対象の Renderer をドラッグ＆ドロップして、必要な項目だけ有効にする流れになります。

AudioLink Reactive is a Unity tool designed to make lilToon AudioLink settings easier to manage in one place. The basic flow is: add the Controller, drag and drop target renderers, then enable only the features you need.

このツールでできること What this tool does

複数 Renderer 管理、非破壊マテリアル複製、Preview、発光テクスチャ、頂点リアクティブを1つの流れで扱えます。

It lets you handle multi-renderer management, non-destructive material duplication, preview, emission texture, and vertex reactive features within one workflow.

向いている人 Who it is for

lilToon を直接細かく触る前に、まず AudioLink リアクティブを導入しやすくしたい人に向いています。

It is suited for users who want an easier entry point into AudioLink reactive setups before working directly with every lilToon field.

基本の流れBasic Flow

最初は次の順番で進めると分かりやすいです。

This order is the easiest way to get started.

  1. 1. AudioLink Reactive Controller を追加する
  2. 1. Add AudioLink Reactive Controller
  3. 2. Hierarchy から対象のオブジェクトをドラッグ＆ドロップする
  4. 2. Drag and drop the target object from the Hierarchy
  5. 3. 一覧に出たマテリアルごとに、発光テクスチャや頂点を有効にする
  6. 3. Enable Emission Texture or Vertex for each listed material
  7. 4. Preview を見ながら強さや色を調整する
  8. 4. Adjust strength and color while checking the Preview
ポイント Tip

最初は 1つの Renderer、1つのマテリアルだけで試すと挙動を把握しやすいです。

Start with one renderer and one material first. It makes the behavior much easier to understand.

発光テクスチャEmission Texture

発光テクスチャでは、音に反応して光る模様や色を設定します。基本は「どの音に反応するか」「どんな模様で光るか」「どんな色で光るか」を決めるイメージです。

Emission Texture controls the glow pattern and color that react to audio. The main idea is deciding which band reacts, what pattern glows, and what color it glows with.

  • 帯域: どの音域に反応させるかを決めます。
  • Band: Chooses which audio band drives the effect.
  • 色 / マスク: 光る模様の元になるテクスチャです。
  • Color / Mask: Texture used as the glowing pattern source.
  • 発光色: 光る色の基準です。
  • Emission Color: The base color of the glow.
  • 合成モード: 光り方の重ね方です。通常は加算を基準に考えると分かりやすいです。
  • Blend Mode: Controls how the glow is combined. Add is usually the easiest starting point.
  • 点滅の強さ: 発光の明滅感を足したい時に使います。
  • Blink Strength: Adds blinking intensity to the glow.
最初のおすすめ Recommended first setup

まずは 帯域 / 色・マスク / 発光色 / 合成モード だけで組むと分かりやすいです。

Start with just Band, Color/Mask, Emission Color, and Blend Mode.

詳細設定 Advanced details

Tiling、Offset、角度、スクロールなどは、模様の見え方を詰めたい時に触ります。

Use Tiling, Offset, Angle, and Scroll when you want to fine-tune how the pattern looks.

注意 Note

最初から全部の項目を触るより、少ない項目で見た目を決めてから詳細に入る方が調整しやすいです。

It is usually easier to decide the overall look with a few key fields first, then move into advanced settings afterward.

頂点の動きVertex

頂点では、音に反応してメッシュがふくらむ動きを設定します。まずは「ふくらみの強さ」だけを見るのがおすすめです。

Vertex settings control mesh inflation driven by audio. As a starting point, it is best to focus only on the inflation strength.

  • 音域: どの帯域で動かすかを決めます。
  • Band: Chooses which audio band drives the motion.
  • ふくらみの強さ: どれだけ膨らませるかの主な項目です。
  • Inflation Strength: Main field controlling how much the mesh inflates.
  • サイズ自動補正: オブジェクトサイズに応じて強さを補正します。
  • Auto Size Correction: Adjusts strength based on object size.
ポイント Tip

慣れるまでは moveDirection などの細かい値は触らず、基本項目だけで見た目を決める方が分かりやすいです。

Until you get used to it, it is easier to ignore detailed values like moveDirection and build the look with only the main fields.

Preview

Preview は Editor 上で最大変形や発光の見え方を確認するための機能です。元のマテリアルを直接壊さず、一時的な複製マテリアルで表示します。

Preview is used to check the maximum deformation or glow appearance in the Editor. It works with temporary duplicated materials instead of directly modifying the original materials.

  • プレビュー有効: Preview を使うかどうかを切り替えます。
  • Preview Enabled: Toggles Preview on or off.
  • プレビュー量: 0 から 1 の範囲で見え方を確認できます。
  • Preview Amount: Lets you preview the effect from 0 to 1.
  • 最大プレビュー: 最大状態をすぐ確認したい時に使います。
  • Max Preview: Quickly checks the maximum state.

よくある質問FAQ

Q. まずどこだけ触ればよいか

Q. What should I touch first?

発光テクスチャなら 帯域 / 色・マスク / 発光色 / 合成モード、頂点なら 音域 / ふくらみの強さ だけで十分です。

For Emission Texture, start with Band, Color/Mask, Emission Color, and Blend Mode. For Vertex, start with Band and Inflation Strength.

Q. lilToon を直接触るのと何が違うか

Q. How is this different from editing lilToon directly?

複数対象管理、非破壊複製、Preview、将来的なメニュー導線まで含めて、導入と運用をまとめることが目的です。

The goal is to unify setup and operation, including multi-target management, non-destructive duplication, preview, and future menu workflow support.

Q. 詳細設定はいつ触るべきか

Q. When should I touch advanced settings?

基本項目だけで大まかな見た目を作ってからで十分です。最初から全部触る必要はありません。

After the overall look is built with the main fields. There is no need to touch everything from the start.
