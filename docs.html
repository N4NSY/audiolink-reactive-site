はじめかたGetting Started

  • Unity上部メニューの Tools / N4NSY / AudioLink Reactive から専用Windowを開きます。
  • Open the dedicated window from Tools / N4NSY / AudioLink Reactive in Unity.
  • Avatar欄に対象Avatarを指定します。指定時に必要なセットアップが自動で行われます。
  • Assign your target avatar in the Avatar field. Required setup runs automatically.

基本の流れBasic Flow

  1. レンダラー一覧から Avatar 配下の Renderer を選びます。
  2. Select a renderer under the avatar from the renderer list.
  3. 選択した Renderer を ALR 対象に追加します。
  4. Add the selected renderer as an ALR target.
  5. 対象Materialは Renderer名 / Material名 で表示されます。
  6. Target materials are shown as Renderer Name / Material Name.
  7. Materialごとに設定し、必要なら保存/読み込みを行います。
  8. Configure each material, then save/load settings when needed.

光らせるGlow

反応の仕方で「光らせる」を選ぶと、音に合わせた発光表現を調整できます。

Choose “Glow” in reaction mode to tune audio-driven emission behavior.

膨らませるInflate

反応の仕方で「膨らませる」を選ぶと、音に合わせたメッシュ変化を調整できます。

Choose “Inflate” in reaction mode to tune audio-driven mesh deformation.

Preview

  • プレビュー有効 / プレビュー量 を使って見た目を確認します。
  • Use Preview Enabled / Preview Amount to check visuals.
  • 表示がおかしい場合は更新ボタンで再同期します。
  • If display looks wrong, press Refresh to resync.

FAQ / TroubleshootingFAQ / Troubleshooting

  • Renderer一覧が空の場合: Avatar欄の指定を確認し、更新ボタンを押してください。
  • If renderer list is empty: verify Avatar assignment and press Refresh.
  • 対象Materialが出ない場合: 先にRendererをALR対象へ追加してください。
  • If no target material appears: add a renderer to ALR targets first.
  • 保存/読み込みはMaterial単位です。
  • Save/Load works per material.
